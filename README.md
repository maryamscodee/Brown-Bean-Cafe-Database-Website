# Brown-Bean-Cafe-Database-Website
Brown Bean Cafe Website &amp; Database System is a full-stack web application developed for online food ordering.
The system allows customers to browse menu items, place orders, and interact with a responsive user interface. The backend is powered by Node.js and MySQL for efficient order and data management.

## Features
Responsive user interface
Interactive menu browsing
Online food ordering
Customer order management
MySQL database integration
Dynamic website functionality using JavaScript
